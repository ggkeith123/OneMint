🚀 UPDATED: Now with ONE-CLICK MINTING!

Users just click a button - no manual USDC sending needed!

═══════════════════════════════════════════════════════════════════════════════

⚡ STEP 1: Install
═══════════════════════════════════════════════════════════════════════════════

cd x402rocks-complete-package
npm install

═══════════════════════════════════════════════════════════════════════════════

🔧 STEP 2: Setup .env File
═══════════════════════════════════════════════════════════════════════════════

cp .env.example .env
nano .env

Paste this (replace with YOUR values):

CONTRACT_ADDRESS=0xYourContractAddressHere
PRIVATE_KEY=0xYourPrivateKeyHere
USDC_PAYMENT_ADDRESS=0xYourWalletAddress
BASE_RPC_URL=https://mainnet.base.org
PORT=3000

Save: Ctrl+X → Y → Enter

═══════════════════════════════════════════════════════════════════════════════

📜 STEP 2A: Deploy Contract (Use Remix)
═══════════════════════════════════════════════════════════════════════════════

1. https://remix.ethereum.org
2. Upload contracts/X402RocksToken-USDC.sol
3. Compile (0.8.20)
4. Deploy to Base with:
   - _usdcAddress: 0x833589fCD6eDb6E08f4c7C32D4f71b54bdA02913
   - _paymentReceiver: Your wallet
5. Copy contract address → Put in .env

═══════════════════════════════════════════════════════════════════════════════

✅ STEP 3: Test Locally
═══════════════════════════════════════════════════════════════════════════════

npm start

Visit: http://localhost:3000

Should see dashboard with "One-Click Mint" button!

═══════════════════════════════════════════════════════════════════════════════

🌐 STEP 4: Push to GitHub
═══════════════════════════════════════════════════════════════════════════════

git init
git add .gitignore .env.example package.json server-automatic.js public/ contracts/ docs/
git commit -m "Add x402rocks"
git remote add origin https://github.com/YOUR-USERNAME/YOUR-REPO.git
git push -u origin main

⚠️  .env should NOT be in git status!

═══════════════════════════════════════════════════════════════════════════════

🚂 STEP 5: Deploy to Railway
═══════════════════════════════════════════════════════════════════════════════

1. https://railway.app
2. New Project → Deploy from GitHub
3. Select your repo
4. Go to Variables tab
5. Paste (with YOUR values):

CONTRACT_ADDRESS=0xYourContractAddressHere
PRIVATE_KEY=0xYourPrivateKeyHere
USDC_PAYMENT_ADDRESS=0xYourWalletAddress
BASE_RPC_URL=https://mainnet.base.org
PORT=3000

6. Save → Wait 2 minutes
7. Copy your Railway URL

═══════════════════════════════════════════════════════════════════════════════

🎉 STEP 6: Test Minting!
═══════════════════════════════════════════════════════════════════════════════

NEW: ONE-CLICK MINTING!
────────────────────────────────────────────────────────────────────────────

1. Open your Railway URL
2. Click "Connect MetaMask"
3. Click "🚀 ONE-CLICK MINT (1 USDC)"
4. Approve USDC in MetaMask
5. Confirm mint transaction
6. ✨ Done! Tokens in wallet!

Time: ~30 seconds total!

ALTERNATIVE: Manual Method (Still Works!)
────────────────────────────────────────────────────────────────────────────

1. Click "Get Payment Address"
2. Send 1 USDC to address
3. Wait 60 seconds
4. ✨ Auto-detected and minted!
