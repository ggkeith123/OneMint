🚀 FASTEST PATH TO DEPLOYMENT

═══════════════════════════════════════════════════════════════════════════════

STEP 1: INSTALL (2 minutes)
────────────────────────────────────────────────────────────────────────────

npm install

═══════════════════════════════════════════════════════════════════════════════

STEP 2: CONFIGURE (3 minutes)
────────────────────────────────────────────────────────────────────────────

cp .env.example .env

Edit .env:
CONTRACT_ADDRESS=0xYourDeployedContractAddress
PRIVATE_KEY=0xYourPrivateKey
USDC_PAYMENT_ADDRESS=0xYourWallet
BASE_RPC_URL=https://mainnet.base.org
PORT=3000

⚠️  Don't have a contract yet? Deploy using Remix first (see STEP 2A below)

═══════════════════════════════════════════════════════════════════════════════

STEP 2A: DEPLOY CONTRACT (Optional - 5 minutes)
────────────────────────────────────────────────────────────────────────────

1. Go to https://remix.ethereum.org
2. Upload contracts/X402RocksToken-USDC.sol
3. Compile (Solidity 0.8.20)
4. Connect MetaMask to Base mainnet
5. Deploy with:
   - USDC: 0x833589fCD6eDb6E08f4c7C32D4f71b54bdA02913
   - Payment receiver: Your wallet
6. Copy contract address to .env

═══════════════════════════════════════════════════════════════════════════════

STEP 3: TEST LOCALLY (2 minutes)
────────────────────────────────────────────────────────────────────────────

npm start

Visit: http://localhost:3000

You should see:
✅ "USDC payment monitoring active!"
✅ Dashboard loads
✅ Can connect MetaMask

═══════════════════════════════════════════════════════════════════════════════

STEP 4: DEPLOY TO RAILWAY (3 minutes)
────────────────────────────────────────────────────────────────────────────

A. Push to GitHub (WITHOUT .env!):

git add .gitignore .env.example server-automatic.js public/ contracts/ package.json
git commit -m "Add x402rocks"
git push origin main

B. Deploy on Railway:

1. Go to https://railway.app
2. New Project → Deploy from GitHub
3. Select your repo
4. Go to Variables tab
5. Add all variables from your .env
6. Save → Auto-deploys!

C. Verify:

curl https://your-app.railway.app/health

═══════════════════════════════════════════════════════════════════════════════

STEP 5: TEST MINTING (2 minutes)
────────────────────────────────────────────────────────────────────────────

1. Open your Railway URL
2. Connect MetaMask
3. Click "Get Payment Address"
4. Send 1 USDC from MetaMask
5. Wait 30-60 seconds
6. Tokens automatically minted! ✨

🔍 TROUBLESHOOTING

═══════════════════════════════════════════════════════════════════════════════

Problem: "Contract not available"
Solution: Deploy contract first, add address to .env

Problem: "Monitoring not active"
Solution: Check USDC_PAYMENT_ADDRESS is set in Railway variables

Problem: "Payment not detected"
Solution: 
- Check logs for errors
- Verify BASE_RPC_URL is working
- Ensure wallet has ETH for gas

Problem: ".env in git status"
Solution: Make sure .gitignore exists and includes .env
