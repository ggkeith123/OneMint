💡 KEY CONCEPTS
═══════════════════════════════════════════════════════════════════════════════

Automatic Detection:
Users send USDC → System detects it automatically → Tokens minted
No manual transaction hash submission needed!

USDC Only:
Stable price ($1 = 1 USDC always)
Low fees on Base (~$0.01)
Simple for users

Base Mainnet:
Layer 2 on Ethereum
Much cheaper than ETH mainnet
Fast transactions

x402 Protocol:
HTTP 402 Payment Required
Standard for paid APIs
Makes service discoverable
